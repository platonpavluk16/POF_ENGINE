## ðŸš€ Getting Started (5 Minutes)

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Engine
```bash
python main.py
```

### Step 3: Create Your First Project
The Project Selector will appear:
```
============================================================
  POF ENGINE - Project Selector
============================================================

1. Create New Project
2. Load Existing Project
3. Run Last Project
4. Exit
```

Choose **1** and follow the prompts to create your first game project!

### Step 4: Start Developing
- Edit `scene.json` to add game objects
- Create Python scripts in the `scripts/` folder
- Configure project settings (resolution, colors, FPS)
- Run your game using the built-in editor
